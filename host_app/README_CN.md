# 360度扫描超声上位机

## 快速启动

双击 **`启动上位机.bat`** 即可启动程序

## 环境要求

- Python 3.8 或更高版本
- 依赖包（见 requirements.txt）

## 启动脚本

| 脚本 | 说明 |
|--------|------|
| `setup.bat` | 首次使用时运行，安装依赖 |
| `启动上位机.bat` | 启动主程序 |

## 功能说明

### 扫描模式 (Scan)
- **配置扫描**: 设置步进角、声速、介质类型
- **开始扫描**: 执行360度全扫描
- **保存CSV**: 导出扫描数据

### 调试模式 (Debug)
- **单点测量**: 在当前角度测量一次
- **移动到角度**: 控制步进电机到指定角度
- **调试扫描**: 在指定角度范围内扫描

### 通信配置

| 参数 | 说明 |
|------|------|
| 端口 | 选择串口 (COM1~COM16 或 SIMULATOR) |
| 波特率 | 115200 bps |
| 数据位 | 8 |
| 停止位 | 1 |
| 校验位 | 无 |

### 设备控制

| 命令 | 说明 |
|--------|------|
| 读取信息 | 获取固件版本、硬件版本 |
| 回零 | 电机回到0度位置 |
| 停止 | 停止当前操作 |

## 硬件连接

```
下位机 (STM32H743)        上位机 (PC)
    PA10 (TX) ─────────────► RX
    PA9  (RX) ◄────────────── TX
    GND ─────────────────────────── GND
```

- 波特率: 115200
- 数据格式: 二进制协议 (0x55AA帧头 + CRC16)

## 协议说明

### 帧格式
```
[0x55 0xAA] [版本 命令 序列号 长度] [数据] [CRC16]
```

### 支持命令
| 命令ID | 功能 |
|----------|------|
| 0x01 | 获取设备信息 |
| 0x02 | 回零 |
| 0x03 | 停止 |
| 0x10 | 配置扫描 |
| 0x11 | 开始扫描 |
| 0x12 | 扫描点数据 |
| 0x13 | 扫描完成 |
| 0x20 | 配置调试 |
| 0x21 | 移动到角度 |
| 0x22 | 单次测量 |
| 0x26 | 开始调试扫描 |
| 0x7F | 状态事件 |

## 故障排除

**连接失败**
- 检查串口号是否正确
- 检查下位机是否上电
- 检查TX/RX是否接反

**数据异常**
- 检查波特率是否一致 (115200)
- 检查RS485收发控制
