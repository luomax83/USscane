# 上位机界面与交互规格 v1.0

## 1. 设计目标

上位机不是“串口工具皮肤”，而是任务控制台。

界面设计必须支持：

- 正式扫描任务的完整闭环
- 调试单点与连续调试扫描
- 原始波形与包络波查看
- 数据记录、保存与回放
- 故障定位与报文追踪

## 2. 总体布局

采用四页签结构：

- 扫描页
- 调试页
- 记录回放页
- 系统页

全局保留顶栏状态区。

## 3. 顶栏状态区

固定显示：

- 串口号
- 波特率
- 连接状态
- 设备状态
- 当前角度
- 当前模式
- 最近一次事件
- 记录状态

固定操作：

- 连接
- 断开
- 回零
- 停止
- 开始记录
- 停止记录

## 4. 扫描页

目标：

- 面向正式扫描任务

布局：

- 左栏：扫描参数区
- 中栏：极坐标结果图
- 右栏：结果统计与任务状态
- 下栏：结果表格与任务日志

### 4.1 扫描参数区

控件：

- 扫描步进角输入
- 介质下拉框
- 自定义声速输入
- 保存为默认模板复选框
- 配置扫描按钮
- 开始扫描按钮
- 停止扫描按钮

交互约束：

- 未连接时禁用
- 未回零完成时禁用“开始扫描”
- 配置成功后显示预计点数

### 4.2 极坐标结果图

显示：

- 360 度极坐标点云/折线
- 当前扫描角度高亮
- 无效点灰显

### 4.3 结果统计区

显示：

- 当前任务 ID
- 总点数
- 已完成点数
- 有效点数
- 无效点数
- 最大距离
- 最小距离
- 平均距离
- 任务开始时间
- 任务结束时间

### 4.4 结果表格

列定义：

- 序号
- 角度
- 距离
- 质量
- 有效位
- 时间戳

### 4.5 日志区

显示：

- 关键事件
- 扫描开始
- 回零完成
- 每次异常
- 扫描结束原因

## 5. 调试页

目标：

- 面向单点测量和连续角度调试

布局：

- 左栏：参数与动作区
- 中栏：波形显示区
- 右栏：位置状态与结果卡片
- 下栏：原始报文与任务日志

### 5.1 参数与动作区

分为两组：

- 单点调试组
- 连续调试扫描组

单点调试组控件：

- 目标角度输入
- 调整到位按钮
- 单次测量按钮

连续调试扫描组控件：

- 起始角度输入
- 结束角度输入
- 每步动作角度输入
- 开始连续扫描按钮

采集配置控件：

- 采集类型：原始波形 / 包络波
- 采样点数输入
- 上传距离结果复选框
- 上传波形复选框

### 5.2 波形显示区

主视图：

- 原始波形曲线

副视图：

- 包络曲线

交互：

- 支持缩放
- 支持游标读数
- 支持切换单次波形与连续任务中的任一角度波形

### 5.3 位置状态与结果卡片

显示：

- 当前角度
- 目标角度
- 运动方向
- 到位状态
- 测量状态
- 当前距离
- 当前质量
- 当前波形帧计数

### 5.4 报文与日志区

显示：

- 原始十六进制收发帧
- 解析后的命令摘要
- 错误码
- 状态事件

## 6. 记录回放页

目标：

- 用于历史任务复核和问题定位

布局：

- 左栏：会话列表
- 中栏：结果/波形主视图
- 右栏：元数据与导出
- 下栏：事件时间线

显示与操作：

- 浏览历史会话
- 加载扫描结果
- 加载单点波形
- 加载连续调试会话
- 导出 CSV
- 导出 JSONL
- 导出波形文件

## 7. 系统页

内容：

- 串口设置
- 波特率档位
- 记录目录
- 自动保存策略
- 协议日志开关
- 调试 ASCII 日志开关
- 设备能力与版本信息

## 8. 关键任务流

### 8.1 正式扫描任务流

1. 连接设备
2. 配置扫描参数
3. 等待设备回零完成
4. 开始扫描
5. 接收扫描点结果
6. 接收扫描结束帧
7. 保存并进入结果查看状态

### 8.2 单点调试任务流

1. 配置调试参数
2. 调整到目标角度
3. 设备到位
4. 开始单次测量
5. 接收波形分包
6. 接收距离结果
7. 接收波形结束帧
8. 保存本次任务

### 8.3 连续角度调试任务流

1. 配置调试范围
2. 开始连续调试扫描
3. 每个角度点接收结果
4. 如启用，接收对应波形
5. 接收连续调试结束帧
6. 保存任务

## 9. 状态驱动规则

- 页面按钮的可用状态必须由设备状态和会话状态共同决定。
- 扫描进行中，禁止再次下发扫描配置。
- 波形上传中，禁止切换任务模式。
- 记录中断时，必须给出明确提示。

## 10. 第一版实现优先级

P0：

- 顶栏状态区
- 扫描页
- 调试页
- 日志区

P1：

- 记录回放页
- 参数模板保存
- 任务摘要统计

P2：

- 更完整的结果分析
- 更复杂的波形对比
